# 🧠 SYNAPSERA

**Where AI meets neuroscience.**

SYNAPSERA is a growing collection of interactive neuroscience, signal-processing, machine-learning, and learning-focused web applications.

This repository is the main landing page for the SYNAPSERA project ecosystem. Each project is maintained as a separate repository and linked below.
